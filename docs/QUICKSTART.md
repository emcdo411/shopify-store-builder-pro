# ⚡ QUICKSTART — 5 Minutes

## 1) Requirements
- Node.js 18+
- Shopify Partner account + dev store
- Shopify CLI
- Git

## 2) Install
```bash
npm install
```

## 3) Configure
Copy `.env.example` to `.env` and fill in values.

## 4) Run local scripts
```bash
# validate theme, generate sample
npm run validate && npm run generate:products
```

## 5) Launch
Follow the prompts in `SKILL-PROMPT.md` to run the Claude skill.
